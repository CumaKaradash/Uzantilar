# Icons Folder

This folder should contain the extension icons:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

For development purposes, you can create simple placeholder icons or download from icon libraries.

The icons should follow the dark minimalist aesthetic with the #0f1115 color scheme.

Recommended design elements:
- Scroll arrow symbol
- Crown or sovereignty symbol
- Dark background with white/light accent
- Clean, minimalist design
